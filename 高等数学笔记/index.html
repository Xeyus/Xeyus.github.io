高等数学笔记

思路

根据求解/证的对象选择方法。先看形式、性质再找可能的解。
证明题不会的话就顺着性质往下推，大概率能推出来。

题型

求极限

  1. 通过无穷小的代换（仅乘除时可换），因式分解等化简。可以考虑求导算极限。
  2. 观察是否为$\frac{0}{0}$（或$\frac{∞}{∞}$），若是或者能化成这种形式，就洛必达。
    1. 想接着洛时看看是不是符合形式；
    2. ∞*0:其中一个变分母；
    3. $∞-∞$:恒等变形，通分；
    4. $0^0$，$∞^0$，$1^∞$:幂指函数通用的e的对数次幂化简；
    5. 其中$1^∞$有公式，极限那里说过，不写了。打着太麻烦。
  3. 若不是，可使用极限的四则运算，比较无穷小阶数等。
  4. 若还不行，若为分式，按照最高次泰勒展开；或者使用夹逼定理（常见于类似数列形式的求极限）。
    1. 麦克劳林：$x_0=0$的泰勒。
  5. 以及最后，最基础的ε 。
  6. 注意是否为单侧极限，是否有极限。
等价无穷小

常用的如tanx-sinx$\frac{x^3}{2}$,1-cosx$\frac{x^2}{2}$等。已有，不在这里重复输入（不想打LaTeX语法了）。
而且能用它的地方都可以用泰勒（就是可能有点麻烦）。

求导与微分

  1. 直接考察求导和微分：
    1. 直接求导/微分。一元函数求导=微分。
    2. y=f(x)的反函数求导/微分：将x视作关于y的函数。$\frac{dx}{dy}=\frac{1}{\frac{dx}{dy}}$。
    3. 隐函数求导/微分：两端对x求导，y视作关于x的函数。
    4. 乘除与幂指：两端取对数。
    5. 参数方程：两端取微，多阶则视作(n-1)阶和x=φ(x)的参数方程再次两端取微。
    6. 过高阶导数找规律。
  2. 应用题：
    1. 相关变化率：微分思想。
    2. 求近似值：运用微分的定义公式。
    3. 其他，就题论题。
  3. 是否可导：
    1. 连续不一定可导/可微，可导/可微一定连续。
    2. 看左右导数是否存在且相等。
连续和间断

第一类间断点：可去间断点，仅该点无定义；跳跃间断点，该点两侧存在不同定义。
第二类间断点：振荡间断点，该点附近无限振荡；无穷间断点，该点至少一侧的极限趋于无穷。均为极限不存在。

求零点/f(x)=y在[a,b]的根个数

  1. 先看函数性质。
  2. 观察其加减、附加系数、变量确定其“原函数”。
  3. 单看它不行就用罗尔、柯西、拉格朗日中值定理。
    1. 一个函数，区间端点函数值相等，区间内可导–>罗尔；
    2. 两个函数/一个分式，可以凑出$\frac{f(a)-f(b)}{g(a)-g(b)}$–>柯西；
    3. 一个函数在两点的函数值作差–>拉格朗日。

函数性质

极值、最值

极值点可能为：

  1. 不连续点；
  2. 连续不可导点；
  3. 可导且导数为零点；
    1. 注意三个充分条件。
      最值点加一个区间端点。
凹凸性

切线上方凹，切线下方凸。
或二阶导大于0凹，小于0凸。
或按定义。

可能为不可导点，二阶不可导点，二阶导为零点。

渐近线

先找任意趋于无穷时的水平，再找任意$x_0$的竖直，再找没有水平一端的斜渐近线（$lim_{x \to \infty}\frac{f(x)}{x}=a,lim{x \to \infty}(f(x)-ax)=b$）。

曲率

弧微分

$(ds)^2=(dy)^2+(dx)^2$
所以有：

  1. y=f(x), $ds=\sqrt{1+f’(x)}dx$

  2. 参数方程：
    $\begin{cases}
    x=φ(t) \
    y=ψ(t) \
    \end{cases}$

    $ds=\sqrt{(φ’(t))^2+(ψ’(t))^2}dt$
    极坐标也可视为参数方程。

  3. 参数不同，弧微分不同。

求曲率

$K=|lim_{\delta s \to 0}\frac{\delta \theta}{\delta s}|=|\frac{d\delta}{ds}|$

所以对y=f(x)，曲率为$\frac{|y’’|}{(1+y’^2)^{\frac{3}{2}}}$

对参数方程：
$\begin{cases}
x=φ(t) \
y=ψ(t) \
\end{cases}$
曲率为$\frac{φ’ψ’’-ψ’φ’’}{φ’^2+ψ’^2}^{\frac{3}{2}}$

曲率圆

与y=f(x)相切，在曲线凹向，$R=\frac{1}{K}$的圆。

积分

基本概念

  1. 不定积分：如果F′(x)=f(x)，则F(x)+C是f(x)的不定积分，记作：$\int f(x)dx=F(x)+C$
    物理意义：不定积分可以看作是累积变化的反函数。
  2. 定积分：定积分是曲线下方面积的代数和，用来表示函数在区间[a,b]上的累积变化量。$\int_{b}^{a} f(x)dx$
    几何意义：定积分表示曲线y=f(x)和x轴之间的有符号面积。
    根据牛顿-莱布尼茨公式：$\int_{b}^{a} f(x)dx$=F(a)-F(b)

基本性质

  1. 线性性质：对定积分和不定积分成立。
    线性运算：加减和数乘。
  2. 区间可加性：对定积分成立，区间[a,b]=[a,c]+[c,b]。同理区间颠倒，值取相反数；规定区间上下限相等，定积分值为0。对有限成立。
  3. 奇偶性：
    1. 若f(x)为奇数，积分区间关于原点对称，积分为0——化简被积函数
    2. 若f(X)为偶数，积分区间关于原点对称，$\int_{-a}^{a} f(x)dx=2\int_{0}_{a} f(x)dx$——缩小积分区间
    3. 周期函数任一周期–>[0,T]
  4. 被积函数较大，则积分较大。
  5. 任何连续函数都存在原函数。

向量代数与解析几何

向量

内积（点积）
算法：对应分量相乘后求和，或模长乘余弦夹角。
适用条件：同维向量（如二维/三维）。
几何意义：投影长度乘积，或判断两向量垂直（内积为0）。

外积（叉积）
算法：三维行列式计算，方向由右手法则确定，模长为两向量模长乘正弦夹角。
适用条件：仅三维向量。
几何意义：结果垂直原向量，模长表平行四边形面积，或判断平行（外积为0向量）。

混合积
算法：先计算两向量的外积，再与第三向量点乘，结果为标量。
适用条件：三个三维向量。
几何意义：绝对值表平行六面体体积，混合积为0则三向量共面。

平面

  1. 平面方程：点法式，一般方程，截距式。
  2. 点到平面距离公式：（类似点到直线距离公式）
  3. 利用数量积求平面夹角：（不是这高中天天用啊，法向量内积比模积）
  4. 空间直线一般方程：两平面交线；
  5. 直线点向式和参数方程：可两两化作参数方程，分母为零也没关系，对应参数=（p-q）的q；
  6. 直线夹角：（似曾相识的方向向量内积比模积）
  7. 直线与平面夹角：（似曾相识的sinθ=内积比模积，对了前面两个求都是cosθ，原因易得）

曲面方程

柱面

易得，缺一个变量的基本上都是柱面（一曲线在另一维度无限延伸）。
以及曲面方程可由曲线方程推广学习理解。

二次曲面

椭球面

方程：x²/a² + y²/b² + z²/c² = 1。
几何意义：三维闭合对称曲面，三轴拉伸的球体。
判别：二次项系数同号，常数项为正。

单叶双曲面

方程：x²/a² + y²/b² - z²/c² = 1。
几何意义：单侧无限延伸曲面，形似双曲线旋转体。
判别：两正一负系数，常数项为正。

双叶双曲面

方程：-x²/a² - y²/b² + z²/c² = 1。
几何意义：分离的两片曲面，沿主轴方向开口。
判别：两负一正系数，常数项为正。

*椭圆抛物面

方程：z = x²/a² + y²/b²。
几何意义：碗状开口向上，横截面为椭圆。
判别：二次项系数同号，一次项存在。

双曲抛物面

方程：z = x²/a² - y²/b²。
几何意义：马鞍形，双向相反开口。
判别：二次项系数异号，一次项存在。

二次锥面

方程：x²/a² + y²/b² - z²/c² = 0。
几何意义：双锥面，顶点在原点，沿轴无限扩展。
判别：二次项系数不全同号，常数项为零。

旋转曲面
  1. 旋转曲面通用求法：旋转一周=圆=圆周到旋转轴对应点距离相等，由此可得原m=f(n)–>a=F(b,c)。
  2. 所以有变量替换规则：
    绕X轴旋转：将原方程中的非旋转轴变量（如y）替换为$\sqrt{y^2 + z^2}$，适用于平面曲线在XY或XZ平面上。
    绕Y轴旋转：将原方程中的非旋转轴变量（如x）替换为$\sqrt{x^2 + z^2}$，适用于平面曲线在XY或YZ平面上。
    绕Y轴旋转：将原方程中的非旋转轴变量（如z）替换为$\sqrt{x^2 + y^2}$，适用于平面曲线在XZ或XZ平面上。
  3. 若圆锥曲线不在XY、XZ或YZ平面，需先将曲线平移或旋转到标准平面，再应用上述替换规则。例如：
    1. 平面变换：通过坐标系旋转或平移，将原曲线方程转换到XY/XZ/YZ平面。
    2. 应用旋转规则：按上述替换规则生成旋转曲面方程。
    3. 逆变换：将生成的曲面方程还原到原坐标系。
  4. 对于任意平面内的圆锥曲线绕任意轴旋转，可通过参数方程和旋转矩阵推导：
    1. 原曲线方程化为$r(t)=(x(t),y(t),z(t))$
    2. 绕某轴旋转角θ，用旋转矩阵$R_?(θ)$作用，此处以绕z轴为例：$x’=x(t)cosθ-y(t)sinθ$, $y’=x(t)sinθ+y(t)cosθ$, $z’=z(t)$
    3. 消去参数t和θ，得到$F(x,y.z)=0$
  5. 由上易得：旋转椭圆、双曲线：绕a，缺b，随c。
  6. 旋转圆：……似乎不用说啊。那就顺便说下注意灵活推广结论吧。
  7. 旋转抛物线：z=$\frac{y^2}{a^2}$绕z轴旋转一周得$z=\frac{x^2}{a^2}+\frac{y^2}{a^2}$，绕y轴旋转得$\frac{y^2}{a^2}$
